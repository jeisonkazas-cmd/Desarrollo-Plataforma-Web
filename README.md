# Frontend – Plataforma de Prácticas

Este proyecto corresponde al **frontend** de la plataforma. Está construido con **React + Vite**, por lo que requiere la instalación de dependencias y un backend disponible para funcionar correctamente.

## Requisitos previos

Antes de ejecutar el proyecto, asegúrate de tener instalado:

- **Node.js** (versión recomendada: 18 o superior)
- **npm** (incluido con Node.js)
- **Backend** corriendo en el puerto **3000**  
  El frontend consume esta API, por lo que debe estar activo antes de iniciar el frontend.

## Instalación

Clona el repositorio e instala las dependencias con:

```bash
npm install
```

## Ejecución del proyecto

Una vez instaladas las dependencias, inicia el servidor de desarrollo:

```bash
npm run dev
```

Esto levantará el frontend en un puerto local (generalmente **5173**).

Asegúrate de que el backend esté corriendo simultáneamente en:

```
http://localhost:3000
```
